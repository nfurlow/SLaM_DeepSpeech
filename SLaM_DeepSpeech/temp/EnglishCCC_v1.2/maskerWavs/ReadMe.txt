This folder contains the 3 continuous masker waveforms [SSN.wav BMN3.wav BAB4.wav] from which the masker segments were chosen for any given confusion. The transcription file [BAB4.txt] describes the word utterances contained in the maskers constructed from natural speech. The format of the transcription is detailed below. The sampling frequency of all the masker waveforms is 16 kHz.


Field			Description						Example
------------------------------------------------------------------------------------------
Start	The location of the onset of a given word utterance in samples		1
End 	The location of the offset of a given word utterance in samples		9760
Speaker The ID of the speaker uttering the word, one of [s1 s2 s3 s4]		s2
Word	The orthographic representation of the uttered word				airline
